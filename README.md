GMTFV는 [YoutubeExplode](https://github.com/Tyrrrz/YoutubeExplode) 라이브러리 기반으로 한
유튜브 고화질 동영상 다운로드 프로그램입니다.


## 책임 제한

본 소프트웨어를 사용함으로 인해 발생하는  
모든 문제, 손해, 데이터 손실 등에 대해  
개발자는 어떠한 책임도 지지 않습니다.
